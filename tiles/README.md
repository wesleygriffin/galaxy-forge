# Tile Images

These tile images are mirrored from the [KeeganW/ti4](https://github.com/KeeganW/ti4)
fan project (`public/tiles/ST_*.webp`). Mirrored locally rather than hot-linked so
the app keeps working if the upstream project moves or removes files.

The underlying tile artwork is the property of Fantasy Flight Games / Asmodee
(Twilight Imperium Fourth Edition). These images are used here for non-commercial
fan-tool purposes consistent with how the upstream project uses them.

Coverage: tile IDs 0–82 (Base game + Prophecy of Kings).
Not yet mirrored: Codex factions (Keleres), Thunder's Edge tiles, Fracture systems.
For tiles without an image, the renderer falls back to its own SVG-drawn glyphs.

If the rights holder wants these removed, we'll remove them.
